Calculadora online guía del usuario

La calculadora permite hacer como los simples cálculos aritméticos, tanto calcular los porcentajes, hacer elevación a potencia, extraer la raíz cuadrada. Las operaciones online se hacen por medio del ratón o del teclado numérico. El tamaño de la calculadora online es ajustable y sus parámetros pueden ser guardados para su uso en el futuro.

Las instrucciones detalladas del uso de la calculadora siguen abajo.

Funciones de las teclas

[ 0 ], [ 1 ], [ 2 ], ... [ 9 ] - teclas numéricas estándar;
[ 00 ] - tecla para inserir un cero doble;
[ → ] - borrar el último carácter;
[ +/- ] - cambiar el signo matemático en la pantalla;
[ XY ] - elevar el número X a la potencia Y;
[ √ ] - extracción de la raíz cuadrada;
[ + ] - adición, [ - ] - sustracción, [ х ] - multiplicación, [ ÷ ] - división;
[ % ] - calculación de porcentajes;
[ M+ ] - guardar en la memoria con el signo [ + ];
[ M- ] - guardar en la memoria con el signo [ - ];
[ MR ] - visualizar el contenido de la memoria;
[ MC ] - borrar el contenido de la memoria;
[ AC ] - reinicializar la calculadora y su memoria;
[ C ] - reinicializar la calculadora sin tocar su memoria.

Introducción de las instrucciones usando el teclado

La calculadora permite usar cualquier tecla numérica de la computadora, como las que se sitúan arriba, tanto las que están a la derecha del teclado.

La tecla [Enter] corresponde al signo "igual".
Para borrar el último carácter introducido se usa la tecla [Backspace] (tecla con una flecha).
Para introducir el signo de adición se usa la tecla [ + ] arriba o a la derecha del teclado.
Para introducir el signo de sustracción se usa la tecla [ - ] arriba o a la derecha del teclado.
Para introducir el signo de multiplicación se usa la tecla [ * ] a la derecha del teclado.
Para introducir el signo de división se usa la tecla [ / ] a la derecha del teclado.
Para reinicializar la calculadora se puede usar las teclas [Esc], [Del] de arriba, o [End] del teclado numérico.

Ejemplos de las calculaciones en la calculadora online

Extracción de la raíz cuadrada del número 529: 529 [ √ ]. Resultado - 23.
Elevar el número 3 a la potencia 4: 3 [ XY ] 4 [ = ]. Resultado 81.
Calcular el porcentaje del número: 500 [ х ] 25 [ % ]. Resultado - 125.
Adición del porcentaje al número: 500 [ + ] 25 [ % ]. Resultado - 625.
Sustracción del porcentaje del número: 500 [ - ] 25 [ % ]. Resultado - 375.

Preguntas frecuentes

En la calculadora online 2+2 x 2 es igual a 8, ¿tal vez la calculadora se equivoca?

No, la calculadora calcula correctamente, pero al introducir cada acción matemática (adición, sustracción, etc.) las calcula alternadamente. Fíjese en la pantalla de las acciones corrientes que está a la derecha debajo de la pantalla principal.

Vamos a calcularlo: 2 + 2=4, el subtotal será 4. Luego 4 x 2=8, la respuesta correcta es 8. ¡No hay faltas algunas!

De la historia de la aparición de las calculadoras

El antecesor de la calculadora moderna fue abaco (del latín "abaco" significa "tabla"). Un abaco fue una tabla con unas cavidades donde se movían las marcas de cálculo (unas piedras o pepitas).

Dicen que el primer abaco fue notado en los pueblos de la Babilonia Antigua hace unos 3 mil años A.C. En la Grecia Antigua, el abaco apareció aproximadamente en el siglo V A.C.
